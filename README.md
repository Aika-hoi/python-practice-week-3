# Учебная практика: Неделя 3

Автор: Болатханова А.Б (ПО-2р-5)
Итоги 3-й недели: Frontend и интеграция
На этой неделе бэкенд перестал быть «невидимым» — мы сделали полноценное веб-приложение.

 Создали интерфейс на HTML и CSS и подключили JavaScript, который отправляет запросы к API через fetch()
 Настроили CORS в FastAPI, чтобы сайт мог нормально общаться с сервером и браузер не блокировал запросы
 Перевели проект на async/await — теперь сервер может обрабатывать много запросов одновременно и работает быстрее
 Обновили Docker-настройки: связали FastAPI и PostgreSQL, добавили переменные окружения для безопасного подключения к базе
